# System design workshop

Denne repo indeholder materialer til en workshop, hvor vi beskriver workflows som input til systemarkitektur.

## Kom i gang

1. Åbn workshop-planen: [workshops/workshop-1.md](workshops/workshop-1.md)
2. Brug skabelonen til hvert workflow: [workshops/skabeloner/workflow-skema.md](workshops/skabeloner/workflow-skema.md)
3. Se eksempler på udfyldte workflows:

- [workshops/eksempler/workflow-eksempel-1-vlan-provisionering.md](workshops/eksempler/workflow-eksempel-1-vlan-provisionering.md)
- [workshops/eksempler/workflow-eksempel-2-firewall-regel.md](workshops/eksempler/workflow-eksempel-2-firewall-regel.md)

## Sådan kører vi Workshop 1

Workflowet i dagen er:

1. Alle skriver workflows ind i Teams Planner som opgaver, hver for sig
2. Vi deler os i makkerpar
3. Hvert par tager et workflow ad gangen og udfylder workflow-skemaet
4. Til sidst samler vi en kort liste over systemer og integrationer der går igen
